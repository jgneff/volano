Unauthorized Host

Problem

You are denied access to this chat server because your Internet Protocol (IP) address is not authorized to make connections. One of the following situations may have caused your address to be unauthorized:

  • A monitor recently banned someone connecting to the chat server from your IP address.
  • A monitor recently banned someone connecting to the chat server from the same range of dynamically assigned IP addresses as your address. In this case, your access is temporarily blocked because a monitor banned someone using the same Internet Service Provider (ISP) as you.
  • The server administrator explicitly denied access from your IP address or from a range of IP addresses containing your address.

Solution

Try again later to see whether the ban of your IP address or network block has been removed or has expired.
