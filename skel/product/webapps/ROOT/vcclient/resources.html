Applet Resources

[Home | Administrator Guide]

This page lists all of the resource files used by the VolanoChat, MyVolanoChat, and WebVolanoChat applets. These files can be modified to customize the appearance and behavior of the VOLANO chat applets on your Web pages. See the VOLANO Chat Server Administrator Guide for complete information on customizing your VOLANO chat applets.

Properties

Properties are read by the applets to customize their appearance when loaded. You can modify these properties from any Web browser by using the Applet Configuration servlet or by editing them directly with any plain text editor.

changes.txt
the list of applet properties deleted, modified, and added between VOLANO Chat Server product versions.

Parameters

Parameters are read by the applet you embed in the banner area of the chat room windows or in the logo area of the main VolanoChat applet window.

BannerPlayer.txt
parameters for the sample embedded applet COM.volano.BannerPlayer used to display rotating full banner advertisements in the chat rooms.
LogoPlayer.txt
parameters for the sample embedded applet COM.volano.BannerPlayer used to display rotating vertical banner advertisements in the logo area of the main VolanoChat applet window.
